This Tableau workbook provides interactive visualizations for workforce demographics, salary patterns, and HR insights.

## Dashboard Tabs

### 1. Workforce Snapshot
- Headcount by department
- Gender distribution
- Age distribution
- Employment type breakdown

### 2. Salary Explorer
- Salary distribution (box plot)
- Salary by department (bar chart)
- Salary by job title (tree map)
- Top earners (highlight table)

### 3. HR Insights
- Retirement risk (age 55+)
- Pay equity analysis
- Salary vs age scatterplot
- Department salary spend

## Data Source
- employees.csv
- SQL query exports (optional)

## How to Use
1. Open the `.twbx` file in Tableau Desktop.
2. Refresh the data source.
3. Use filters to explore departments, gender, and job titles.
