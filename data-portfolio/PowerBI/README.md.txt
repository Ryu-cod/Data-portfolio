This Power BI dashboard visualizes employee demographics, salary distribution, and departmental insights using the cleaned SQL dataset.

## Pages Included

### 1. Employee Overview
- Total employees (KPI)
- Gender distribution (donut chart)
- Age distribution (histogram)
- Headcount by department (bar chart)

### 2. Salary Analysis
- Salary distribution (box plot)
- Average salary by department (bar chart)
- Top 10 highest-paid employees (table)
- Salary vs age (scatter plot)

### 3. HR Risk & Equity
- Employees nearing retirement (table)
- Gender pay gap (side-by-side bar)
- Pay equity flags (heatmap)
- Tenure distribution (histogram)

## Data Source
- employees.csv (from the project root)
- SQL outputs from the scripts folder

## How to Use
1. Open the `.pbix` file in Power BI Desktop.
2. Refresh the data source to load the latest CSV.
3. Interact with slicers for department, gender, and job title.
